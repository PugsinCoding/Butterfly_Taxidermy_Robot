# Butterfly_Taxidermy_Robot
Framed Robot Butterfly meant to look like taxidermy that flutters its wings once something gets within a certain distance. Meant to add whimsy in my room.

This is also my first robot project without a major tutorial. I will be writing down what I have learned throughout this experience. I am mainly learning how to use different sources of power besides a 9V battery, specifically a 9V Center Positive Adapter, a power supply module component, a distance sensor, and step motors.
